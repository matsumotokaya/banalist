## 現状整理

- **サービス名**: IMAGINE

- **ポジショニング**: 自分のために開発したツールを一般公開。投稿しているアニメイラスト風の画像がそのままテンプレートとして使える

## Instagramマーケティングアイテム（影響力順）

### 1. リール（最強）

- 動画コンテンツはリーチが最も大きい
- IMAGINEのデモ映像をリール化する（操作の早送り等）
- BGM選びが重要（トレンドBGMで伸びる場合がある）
- 制作コストは高いが見返りも大きい

### 2. フィード投稿（主力）

- このアカウントはフィードで成長してきた実績がある
- 1万いいねを超えるフィードが出ると一瞬でフォロワー増・リーチ拡大
- Episode形式で継続投稿（Episode 428〜）
- 毎回2枚目にIMAGINE告知バナーを添付

### 3. ストーリーズ（未活用 → 活用必須）

- フォロワー向けのエンゲージメント強化枠
- フォロワーにしか届かないが接触頻度を上げられる
- デモビデオ、新テンプレート告知、アンケートなど
- 毎回使えるIMAGINE告知バナーを定型化しておく

### 4. 固定投稿（プロフィールTOP3）

- プロフィール訪問者への訴求力が高い
- **現在**: 3投稿で1つのキービジュアルを構成中 → 今回は変更しない

### 5. ノート

- アバターの吹き出しに短いメッセージを表示
- フォロワーのフィード上部に表示される
- サービス告知を定期的に更新する

### 6. リンクセクション

- 現在はIMAGINEのみリンク中（最低限OK）
- リンク集（Linktree的なもの）にジャンプさせる案も検討
  - IMAGINE / ブランドサイト / STORES / Discord をまとめる

### 7. Threads（20,000フォロワー）

- Instagramと連動した告知・日常発信
- プロフィール欄の更新も必要
- フィード投稿と同時に告知を流す運用

## 継続運用プラン

### 投稿頻度

- 2〜3日に1回（最低週2〜3回）
- Episode 428〜 として連番を継続

### 基本方針

「作品を見せる → IMAGINEで作れると伝える → 使ってもらう」の自然な導線。
投稿コンテンツ = テンプレート商品なので、投稿制作コスト ≒ 商品制作コスト（一石二鳥）。

### 運用効率化

- キャプション・告知バナー・ストーリーズバナーを定型化し、毎回の制作コストを下げる
- BGMは都度選定（トレンドBGMで伸びる可能性があるため手を抜かない）

### 既存サブスクユーザー移行キャンペーン

#### 概要

Instagramサブスクリプション会員向けの移行特典。サブスクを放置していたお詫びも兼ねて、IMAGINEのプレミアムプランを6ヶ月間無料で提供する。対象者は少数のため宣伝効果も期待。

#### 特典内容

- IMAGINEのPremiumプラン（通常 $8/月）を **6ヶ月間無料**
- 6ヶ月経過後は自動的にFreeプランに移行（課金なし、安心）
- 対象: 過去にInstagramサブスクリプションに登録していた方

#### システム対応

手動対応（対象者が少数のため）:

```sql
-- 対象ユーザーのプレミアム化
UPDATE public.profiles
SET
  subscription_tier = 'premium',
  subscription_expires_at = '2026-08-22T00:00:00Z'
WHERE email = '対象ユーザーのメール';
```

- 6ヶ月後（2026年8月頃）に手動で `subscription_tier = 'free'` に戻す
- カレンダーにリマインダーを設定しておく
